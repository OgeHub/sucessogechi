Explore: Blog API

  • Overview: Explore is a scalable RESTful API inspired by platforms like Medium. It powers an application where users can authenticate, create blog posts, interact with content, and subscribe to memberships.

  • Technologies: Node.js, Express, TypeScript, MongoDB, Mongoose, Brevo, Render, Cloudinary, Docker, JWT

  • Key Features: Secure user authentication (JWT), email verification & password reset flows, user management, blog post creation, comments and replies, claps, subscriptions, role-based access control, pagination-ready endpoints, centralized error handling, and API documentation.

  • Engineering Focus: Clean architecture, reusable middleware, request validation, secure password hashing, environment-based configuration, and containerized deployment.

  • GitHub: https://github.com/OgeHub/blog
